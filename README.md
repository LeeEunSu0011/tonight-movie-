# 🎬 오늘 밤 영화

> 오늘 밤 21:30~22:10 사이에 특정 TV 채널에서 방영하는 영화를 한눈에 확인하는 PWA

**배포 주소:** https://leeeunsu0011.github.io/tonight-movie- -> 사용자가 많아지면 도메인으로 변경 예정

---

## 개발 목적

"오늘 밤에 볼 만한 영화가 TV에 있나?" 라는 단순한 질문에서 시작했습니다.  
개발 경험이 있지만 AI로 개발하는건 어떤 느낌을지도 궁금해서 바이브 코딩으로 구현했습니다.
매번 여러 채널 편성표를 뒤지지 않고, 21:30~22:10 사이에 시작하는 **영화만** 모아서 보여주는 서비스입니다.

---

## 주요 기능

- 📺 **영화 전용 필터** - 영화 장르만 수집
- 🕙 **시간대 필터** - 21:30~22:10 시작 프로그램만 표시
- 📅 **오늘/내일 탭** - 오늘과 내일 편성표 전환
- 🎬 **영화 포스터** - TMDB API로 자동 표시
- ⭐ **평점/감독/출연진** - TMDB 상세정보 제공
- 🔞 **연령 배지** - 12·15세 ⚠️, 19세 🔞 표시
- 📝 **줄거리** - 영화 시놉시스 표시
- 🔄 **자동 갱신** - 하루 4회 자동 업데이트
- 📱 **PWA** - 홈 화면에 추가하여 앱처럼 사용 가능

---

## 지원 채널

KBS1, KBS2, MBC, SBS, tvN, OCN, OCN Movies, OCN Movies2

---

## PWA 설치 방법

### 안드로이드 (Chrome)
1. 브라우저에서 사이트 접속
2. 우측 상단 메뉴(⋮) → **홈 화면에 추가**

### iOS (Safari)
1. Safari에서 사이트 접속
2. 하단 공유 버튼(□↑) → **홈 화면에 추가**
